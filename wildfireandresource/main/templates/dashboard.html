Welcome to Your Mission Control

The WildFireandResource App is designed to streamline the management of resource orders and emergency documentation. Our system empowers you to efficiently handle critical wildfire incident forms, ensuring accuracy and rapid deployment.

Whether you're uploading existing resource orders or generating new forms from scratch, our intelligent system minimizes manual data entry and helps you focus on what matters most.

How It Works

Our application simplifies the process of managing critical incident forms through intelligent data extraction and generation.

1. Agreement Contract Processing

  • Upload an Existing PDF: Upload your agreement contract PDF file. The system will prompt you to provide an associated short description. Once uploaded, The application intelligently mines crucial information, including:
    • Contract Number
    • Issuer Contact Information
    • Amount
    • Important Dates
    • Vendor Information
    • Associated Resources
    This data is then securely stored and readily accessible for further operations.

2. Resource Order (RO) Form Management

You can manage Resource Order forms in two ways:

  • Upload an Existing PDF: Upload your Resource Order PDF file. The system will prompt you to select an associated agreement contract. Once uploaded, our intelligent parser will automatically mine the following critical data points directly from the PDF:
    • Project Name
    • Incident Number
    • Resource Model and Make
    • Request Number
    • Resource Assigned
    The form is then securely stored for future reference.
  • Manual Data Entry: Alternatively, if you don't have a PDF to upload, you can manually input the above-mentioned information directly into the system using provided input fields. This flexibility ensures you can generate necessary documentation quickly.

3. Generating Essential Forms

The mined or manually entered data from the Resource Order Form is then used to populate two vital documents:

  • Emergency Shift Ticket: This form utilizes all the information retrieved from the Resource Order Form. Additionally, the system provides extra input fields where you can add more specific details required for the Emergency Shift Ticket.
  • Evaluation Form: The Evaluation form is designed for simplicity. It exclusively uses the information mined (or manually entered) from the Resource Order Form, requiring no additional user input for its generation.

This automated process ensures consistency and reduces errors across your incident documentation.
